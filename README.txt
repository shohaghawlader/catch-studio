# Catch Studio 360 Website

A modern, mobile-friendly static website for Catch Studio 360.

## Files
- `index.html` — main webpage
- `css/style.css` — full responsive styling
- `js/script.js` — gallery, lightbox, calculator, FAQ, WhatsApp booking form
- `assets/images/` — optimized WebP images
- `assets/video/studio-tour.mp4` — studio promo video

## How to use
Open `index.html` directly in your browser, or upload the whole folder to Vercel, Netlify, cPanel, GitHub Pages or any static hosting.

## Booking Form
The booking form opens WhatsApp with all submitted details and sends to: `+8801707680865`.

## Important
Do not rename or move the `assets`, `css`, or `js` folders unless you also update the paths inside `index.html`.
